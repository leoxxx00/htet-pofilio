Agentic Q&A Web Crawler

Developed a production-oriented agentic application that analyzes imported URLs and produces high-quality training pairs. The system was built using FastAPI, Next.js, HTML, and CSS, with Docker enabling consistent containerized deployment. Hosted on AWS, the platform incorporates load balancing, secure domain and traffic management, and HTTPS enforcement to support reliable and secure cloud delivery. PostgreSQL and Redis were implemented for RAG data storage and caching, helping improve performance and scalability. This project highlights expertise in building agentic systems from prototype through to full-stack development, containerization, and secure SaaS deployment in production.

How to use it: 1. Enter a URL 2. Click Evaluate 3. Click Generate 4. Download the generated Q&A pairs. Processing time depends on the size and complexity of the content.
