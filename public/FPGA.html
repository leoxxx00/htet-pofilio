Getting Started with FPGA

Introduction

Here are the essential steps and notes to begin programming FPGAs using Xilinx Vivado on the BASYS 3 board.

Important Notes

  • Know your FPGA chip and exact model number.
  • Understand the purpose of jumper settings.
  • Use a proper data-transfer USB cable (not a charging-only cable).

Steps to Program an FPGA

  1. Create a new Vivado project.
  2. Select the correct FPGA chip/model (e.g., Artix-7 XC7A35T-1CPG236C).
  3. Use consistent naming for your top module.
  4. Add your design source files (VHDL or Verilog).
  5. Create or add your constraints file (XDC) for pins.
  6. Run Synthesis.
  7. Run Implementation.
  8. Generate Bitstream.
  9. Open Hardware Manager → Open Target → Auto-connect.
  10. Program Device with the generated bitstream.
