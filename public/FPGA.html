FPGA Programming: Digital Lock System (Verilog)

A digital lock system implemented using Verilog on an FPGA board. This project demonstrates:

• Parallel processing using shift registers • Real-time input handling • Synchronous logic design • Hardware-level digital security concepts

FPGAs serve as the backbone of modern systems in telecom, AI hardware, aerospace, and more. This design uses clocked logic and modular coding to build a reliable lock system.
